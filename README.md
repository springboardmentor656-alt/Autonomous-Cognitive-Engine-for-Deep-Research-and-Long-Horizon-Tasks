## Overview
This project implements a LangGraph-based planning agent that decomposes complex
user requests into structured TODO lists.

## Tech Stack
- Python 3.11+
- LangGraph
- LangChain
- OpenAI GPT-4o-mini
- LangSmith (Tracing)

## How to Run
Create a .env file with:
   OPENAI_API_KEY=your_key
   LANGSMITH_API_KEY=your_key
   LANGSMITH_TRACING=true
   LANGSMITH_PROJECT=milestone-1-planner
